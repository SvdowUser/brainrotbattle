Drop your street-lamp (Laterne) .glb models here as:
  lantern1.glb, lantern2.glb, lantern3.glb, ... up to lantern10.glb

One lantern model is enough to start — you'll see it repeat along the street.
Add more files later and the game will mix them in automatically on restart.

Each lamp is auto-scaled to ~5m tall and grounded. Source orientation doesn't
matter; the game will point the arm toward the road.

Format: GLB (binary glTF 2.0). Any free city-pack lamp works (Sketchfab /
Poly Pizza / Kenney).
