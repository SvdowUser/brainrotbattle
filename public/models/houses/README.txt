Drop your house/building .glb models here as:
  house1.glb, house2.glb, house3.glb, ... up to house20.glb

The game looks for these filenames automatically — just drop them in and
restart the dev server (npm run dev). If a file is missing the game simply
skips it.

Each house is auto-scaled to ~12m tall and grounded, so any source size works.
These are spawned on both sides of the road behind the sidewalk.

Format: GLB (binary glTF 2.0). Download from Sketchfab / Poly Pizza / Kenney.
