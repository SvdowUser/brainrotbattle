Drop your palm tree .glb models here as:
  palm1.glb, palm2.glb, palm3.glb, ... up to palm20.glb

The game looks for these filenames automatically — just drop them in and
restart the dev server (npm run dev). If no palms are present the game
falls back to the procedural palm (cylinder trunk + fan fronds).

Each palm is auto-scaled to ~4.5m tall and grounded, so any source size
works. Palms spawn along the sidewalk in the coastal biome (the second
biome, after ~1200m of running).

For the "3 Varianten"-starter set, name them:
  palm1.glb  palm2.glb  palm3.glb

You can add more later (up to palm20.glb); existing files are never
deleted when you add more.

Format: GLB (binary glTF 2.0). Download from Sketchfab / Poly Pizza /
Kenney as .glb, not .gltf.
