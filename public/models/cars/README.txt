Drop your .glb car models here as:
  car1.glb, car2.glb, car3.glb, ... up to car20.glb

Just drop them in and restart the dev server (npm run dev) — the game
discovers new files automatically. Existing files are never deleted when
you add more.

Each car is auto-scaled to ~4m length and grounded, regardless of source
size/orientation. If a file is missing, it's just skipped — no crash.

Format: GLB (binary glTF 2.0). Download from Sketchfab / Poly Pizza /
Kenney as .glb, not .gltf.
