Drop your long blocker .glb models here as:
  bus1.glb, bus2.glb, bus3.glb, ... up to bus20.glb

(An un-numbered bus.glb is also accepted.)

These replace the red subway-wagon obstacle. They must be LONG enough that
the player can't jump over them — e.g. bus, lorry, truck. Any length is fine
at source; the game auto-scales to ~10m long.

Drop a file in, restart the dev server, and it's picked up. Existing files
are never deleted.

If a file is missing, the game falls back to the procedural red wagon.
